# Hyde Agent v1.0 - Predefined Capabilities & Commands Guide

Hyde Agent is a lightweight, keyboard-and-voice-driven desktop assistant built using Tauri, Rust, and Python. It functions as a **deterministic, rule-based command overlay** (activated globally via `Alt + Space`). 

Rather than using complex generative AI models to interpret instructions, it processes inputs using **exact matching, predefined aliases, string prefix rules, and basic fuzzy/regex matching** to execute native OS actions and return predefined responses.

## рџЋ™пёЏ Core Interaction Modes

### 1. Voice Command Mode
Hyde Agent transcribes voice instructions using a native Python sidecar.
* **Transcription Engine:** Uses Python's `speech_recognition` module powered by the **Google Speech API** to transcribe spoken audio.
* **Deterministic Matching:** Once transcribed, the text is run through the exact same parsing rules as typed input.
* **Real-time Status Updates:** The overlay status bar updates with the following states:
  * `Initializing Microphone...`
  * `Listening... Speak now.` (Ready state)
  * `Speech detected... Processing...` (Processing state)
  * `Timeout: No speech detected.` (Safely cancels on silence)
  * `Error recognizing speech.` (Fails gracefully)

### 2. Chat Overlay Mode
A search-and-execute input overlay bar.
* **Global Hotkey:** `Alt + Space` toggles the overlay window.
* **Auto-Suggestions:** As you type, the dropdown dynamically matches keywords from the predefined command registry.

## рџ› пёЏ Predefined Command Registry

Hyde Agent handles queries strictly using the following predefined rules:

### 1. App Launching (Predefined Apps & Fallbacks)
Launches local apps using predefined aliases.
* **Supported Local Apps:** Matches `brave`, `chrome`, `firefox`, `edge`, `notepad`, `vscode`, `terminal` (Command Prompt/PowerShell), `explorer` (File Explorer), `calculator`, `spotify`, `discord`, `steam`, `task manager`, `settings` (OS Settings), `paint`, `word`, `excel`, `powerpoint`, `telegram`, `slack`, `obs`, `vlc`, and `zoom`.
* **Windows Search Fallback:** If the app name is not directly matched, it runs a PowerShell command to search the Windows Start Menu apps (`Get-StartApps`) and open it.
* **Web Search Fallback:** If the app cannot be found locally or via Windows search, it automatically opens Google Search for the app name in your default browser.
* **Examples:** *"open notepad"*, *"launch vscode"*, *"open paint"*

### 2. Predefined Websites
Opens the default browser directly to specific URLs.
* **Supported Sites:** YouTube (`youtube`), GitHub (`github`), Google (`google`), Gmail (`gmail`), Yahoo Mail (`yahoo mail`), Outlook (`outlook`), Reddit (`reddit`), Twitter/X (`twitter`), Instagram (`instagram`), WhatsApp Web (`whatsapp web`), ChatGPT (`chatgpt`), Claude (`claude`), Netflix (`netflix`), Amazon (`amazon`), Facebook (`facebook`), LinkedIn (`linkedin`), Twitch (`twitch`), Pinterest (`pinterest`), StackOverflow (`stackoverflow`), Wikipedia (`wikipedia`), eBay (`ebay`), Spotify Web (`spotify web`), Discord Web (`discord web`), Figma (`figma`), Notion (`notion`), Canva (`canva`), Medium (`medium`), Quora (`quora`), Imgur (`imgur`), Bing (`bing`), Yahoo (`yahoo`), DuckDuckGo (`duckduckgo`), Weather (`weather`), Maps (`maps`), Calendar (`calendar`), Drive (`drive`), Photos (`photos`), Translate (`translate`), News (`news`), Meet (`meet`), Zoom Web (`zoom web`), TikTok (`tiktok`), Snapchat (`snapchat`), Vimeo (`vimeo`), SoundCloud (`soundcloud`), Apple Music (`apple music`), Hulu (`hulu`), Disney Plus (`disney plus`), HBO Max (`hbo max`), and Prime Video (`prime video`).
* **Examples:** *"open github"*, *"go to chatgpt"*, *"open netflix"*

### 3. System & Power Controls
Direct control over specific system states.
* **Volume Controls:** Volume Up (increases volume step-by-step), Volume Down (decreases step-by-step), and Mute (toggles sound).
* **Workstation Lock:** Lock screen instantly.
* **Power Control:** Put Windows to sleep (`sleep`), restart PC (`restart`), or shut down PC (`shutdown`). *Note: Shutdown/Restart commands execute a 5-second countdown warning before running.*
* **Examples:** *"volume up"*, *"mute"*, *"lock screen"*, *"shutdown"*

### 4. Web & YouTube Searching
Performs browser-based search queries via URL parameters.
* **Web Search:** Matches prefix "search [query]", "google [query]" or "look up [query]" to search Google.
* **YouTube Search:** Matches prefix "play [query] on youtube", "search youtube [query]", or "youtube [query]" to search YouTube.
* **Email Search:** Matches Regex queries like "search mail from [sender]" or "email about [topic]" to open Google Mail search.
* **Examples:** *"google rust programming"*, *"play lo-fi music on youtube"*, *"search mail from boss"*

### 5. File & Folder Operations
Opens files and directories on your local drive.
* **File Access:** Matches prefix "open file [absolute file path]".
* **Folder Access:** Matches prefix "open folder [absolute folder path]".
* **Examples:** *"open folder C:\Users"*, *"open file D:\projects\notes.txt"*

### 6. Timers & Reminders
Provides countdown timer functionality.
* **Timer Trigger:** Matches prefix "set timer [X] minutes" or "timer [X] minutes". The Rust backend spawns a task that sleeps for the duration, triggers a system beep sound, and sends a notification toast.
* **Examples:** *"set timer 5 minutes"*

### 7. Clipboard Actions
Simple string clipboard helpers.
* **Copy Text:** Matches prefix "copy [text]" to copy the text to your system clipboard.
* **View Clipboard:** Matches "show clipboard" (or "clipboard", "clip history") to retrieve and display the current string stored in your clipboard.
* **Examples:** *"copy Hello World"*, *"show clipboard"*

### 8. System Information
Retrieves specific telemetry metrics using native crates.
* **Time:** Current local system time (e.g., `10:30 AM`).
* **Date:** Full current date (e.g., `Saturday, June 20, 2026`).
* **Local IP:** Internal IP address of your device.
* **RAM Memory:** Current MB of used memory vs total system memory.
* **Battery:** Current charge percentage and battery state (e.g., `85% (Discharging)`).
* **Examples:** *"what time is it"*, *"battery"*, *"ram"*, *"my ip"*

## рџ’¬ Predefined Small Talk & Chat Responses

When conversational phrases are inputted, Hyde Agent returns simple, hardcoded answers:

| Predefined Input Pattern / Greeting | Predefined Response Text |
| :--- | :--- |
| `"hi"` / `"hello"` / `"hey"` / `"hallo"` / `"hiya"` | `"Hello! How can I help you today?"` |
| `"how are you"` / `"how's it going"` | `"I'm functioning perfectly, thank you for asking!"` |
| `"who are you"` / `"what are you"` | `"I'm Hyde Agent, your personal desktop assistant."` |
| `"good morning"` / `"morning"` | `"Good morning! Ready to tackle the day?"` |
| `"good evening"` / `"evening"` | `"Good evening! Need help winding down?"` |
| *Any other conversational inputs (e.g. "goodnight", "thank you", "bye")* | `"Hello there!"` (Default Fallback) |

# Hyde Agent V3 Intents

This file enumerates the standardized Intents generated by the Neural Intent Parser and consumed by the V3 Router.

## Application & OS Level
| Intent Name       | Parameters Expected | Example Matches |
|-------------------|---------------------|-----------------|
| `OPEN_WEBSITE`    | `url`               | "open google.com", "open thenn.in" |
| `OPEN_APP`        | `app_name`          | "open chrome", "launch visual studio" |
| `CLOSE_APP`       | `app_name`          | "close spotify", "kill chrome" |
| `SLEEP_PC`        | None                | "sleep the pc", "suspend my computer" |
| `SHUTDOWN_PC`     | None                | "shutdown the pc", "turn off the computer" |
| `RESTART_PC`      | None                | "restart the pc", "reboot" |
| `LOCK_PC`         | None                | "lock my screen" |
| `SCREENSHOT`      | None                | "take a screenshot" |
| `EMPTY_TRASH`     | None                | "empty the recycle bin" |

## AI Tasks (Browser & API Mode)
| Intent Name       | Parameters Expected | Example Matches |
|-------------------|---------------------|-----------------|
| `AI_CHAT`         | `query`, `ai_type`  | "tell me a joke", "explain quantum computing", "research rome" |
| `AI_GENERATION`   | `query`, `ai_type`  | "write a resignation letter", "design a logo concept" |
| `SMALL_TALK`      | `message`           | "how are you", "good morning" |

## Scheduling & Info
| Intent Name       | Parameters Expected | Example Matches |
|-------------------|---------------------|-----------------|
| `SET_TIMER`       | `minutes`           | "set a timer for 5 minutes" |
| `SET_REMINDER`    | `message`, `delay_minutes` | "remind me to call mom in 10 minutes" |
| `LIST_TIMERS`     | None                | "what are my timers" |
| `CANCEL_TIMER`    | None                | "cancel my timer" |
| `TIME_QUERY`      | None                | "what time is it" |
| `DATE_QUERY`      | None                | "what is today's date" |
| `SYSTEM_INFO`     | `type`              | "how much ram am i using", "battery level" |

## Fallbacks
| Scenario | Fallback Action |
|----------|-----------------|
| Unrecognized Pattern | **AI_CHAT** (Routes to selected AI Provider) |
| Missing API Key      | **Browser AI Mode** (Opens URL deep-link to AI Provider) |
